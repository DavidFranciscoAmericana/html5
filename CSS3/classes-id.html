Classes e IDs

O id é um identificador único ele identifica um ponto único, não é permitido ter dois ids de nomes identicos, uma classe já é possivel utilizar diversas vezes, ou seja, utilizando uma classe pode definir vários elementos com os mesmos atributos dessa classe. Esse arquivo em uso de nome inline_style estou utilizando um estilo no head, agora estarei criando um arquivo chamado de classes-id.html para separar a estilização do arquivo.

Para utilizar uma classe usamos: ponto + nome da classe + {} e definimos as propriedades dessa classe, para chamar essa classe usamos na tag class="nomeDaClasse", se utilizarmos uma determinada estilização de cor padrao e quizermos alterar no projeto inteiro, modificamos as propriedades de classe e todas as tags que utilizam a classe serao afetadas.

Para definir um id usamos # + nome único + {} e entre as chaves, o id é unico em uma página,

Div e Span

Usamos a div para dividir e estruturar os sites, é uma forma de organizar os elementos de uma página Web, Modelo de div clique nesse link e verifique um modelo de div e span, a div agrupa os elementos na vertical o span agrupa os elementos inline, ou seja, na horizontal .

Fontes e cores

Nesse paragrafo vou utilizar uma classe por mim criada de formatação onde estarão as propriedades de fontes e cores, assim ficara diferente dos demais paragrafos. A cores pode ser usadas o nome em ingles ou utilizar o código hexadecimal.

Já as fontes podem ser alteradas os tamonhos, tipo, familia, em familia utilizamos o nome da fonte entre aspas dupla e qual familia pertence, em seguida outro tipo de fonte e em seguida a familia, caso o browser não encontre a a primeira, tenta a segunda e caso tambem não encontre vai pra terceira opção: font-family: "Times New Roman" , Times , serif;
