Padding

Padding é todo o espaçamento interno, que é adicionado a uma caixa, para ver esse exemplo funcionando é só colocar uma borda no componente em que o padding será usado.

Margin

Margin é todo o espaçamento externo, que é adicionado a uma caixa, para ver esse exemplo funcionando é só colocar uma borda no componente em que a margim será aplicada.
