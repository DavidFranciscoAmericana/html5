Formulários

São usados para autenticação de usuário, cadastro em geral e como forma de enviar contatos, para isso usamos a tag "form", "input" para receber os dados digitados pelo usuário, e um "button" para poder enviar o que foi digitado. O type varia conforme o tipo de dados que será inserido ou ação que será efetuada, ex type="submit" envia os dados para onde foi programado.
