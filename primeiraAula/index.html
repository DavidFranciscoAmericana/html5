Imagens

Para inserir imagens no projeto crie uma pasta de nome imagens e nela adicione as imagens necessárias, pode inserir o url externo para buscar imagens em sites e diretorios, para isso use a tag "img" e no url adicione o endereço e nome da imagem necessária. Pode definir largura e altura utilizando "width ou hight" se utilizar ambos você deve fazer de forma proporcional exemplo a imagem possui 300x900 e você deseja diminui-la faz 150x450 ou utilize apanas hight="450" ou apenas "width="150" assim altera proporcionalmente sem causar distorsão, quanto ao risco de usar imagens de url da Web o risco é se o dono do site indisponibilizar a imagem ou mudar de endereço você não conseguirá exibir essa imagem
