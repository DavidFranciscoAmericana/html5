Tabelas

As tabelas são usadas para tabular dados em uma página web, são utilizadas praticamente três tags "table", "tr" table row e "td" que são os dados da tabela, onde cada tr é uma linha da tabela e cada td é uma coluna, utilizando a primeira th com isso os titulos dos dados ficam destacados. Podemos usar alguns atributos como (border, width, height,), é possível colocar imagens, links, pode utilizar width com porcentagem ex 100% sempre vai ocupar o total da pagina, 70% vai ocupar 70% da pagina e assim por diante. temos tambem o colspan="2" nesse caso 1 unica coluna ocupa o espaço de 2 colunas, caso estivesse o 3 no colspan seria o espaço de 3 colunas e temos o rowspan que é semelhante ao colspan, mas ao inves de agrupar colunas agrupa linha
